Download MALS Library
Mobile app licensing library is an android archive library. Any android app can be wrapped using this library as plug and play. It provide licensing & its security for all their registered user.
Download AAR Library Download How to use:

1. Add MAL library file(*.aar) in you application libs folder.

2. In build gradle file add required library dependencies as shown below:

> compile (name: 'myaarlibrary-release', ext:'aar')

3. Access library class any where in application as per below references along with required parameters:

> To create license: new MobileLicense().createLicense()

> To verify license: new MobileLicense().verifyLicense()
